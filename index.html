About

The LilyBukkit Project aims to provide a toolchain for Minecraft 16.05 server hosting. Here are our projects:
  • LilyBukkit-API - API library for plugin development, fork of Bukkit
  • LilyBukkit-Core - server core for 16.05 versions
  • Nebula - a server portal suite and a proxy for LilyBukkit-Core
  • Alphadax - a Minecraft protocol parser, fork of md-5's Mendax

LilyBukkit-Core features

OPTIMIZATION

LilyBukkit combines CraftBukkit's optimization with Rosepad's fixes. It makes the server run more stable, provides a better gameplay experience for the players and allows server owners to debug errors easier.

BUKKIT

The LilyBukkit project provides a fork of Bukkit API for 1.0.16.05_13 Lilypad QA. On top of that, LilyBukkit-Core provides a compatibility level for plugins written for CraftBukkit 1060, making it possible for server owners to install plugins they like without worrying about changes in the API.

Note: Some plugins, like Essentials, may still require porting as they are heavily version-dependent.

RML

LilyBukkit-Core provides Rosepad ModLoader support out of the box, allowing the server owners to customize the players' gameplay experience at their will.

Supported versions

  • Alpha 1.0.16.05 Unr.Preview2 (upcoming)
  • Alpha 1.0.16.05_13 Lilypad QA (Rosepad is recommended, although LPUJ and Vanilla should work just fine)
