Optimization

LilyBukkit combines bits of CraftBukkit code with Rosepad's fixes. It makes the server run more stable, provides a better gameplay experience for the players and allows server owners to debug errors easier.

Bukkit plugins

The LilyBukkit project provides a proper API to create plugins for 1.0.16.05_13 Lilypad QA. On top of that, LilyBukkit-Core provides a compatibility level for plugins written for CraftBukkit 1060, making it possible for server owners to install plugins they like without worrying about changes in the API.

Note: Some plugins, like Essentials, may require porting as they are heavily version-dependent.

RML

LilyBukkit-Core provides out-of-box support for RML modifications.
